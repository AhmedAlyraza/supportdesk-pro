# SupportDesk Pro

A browser-based help desk dashboard built with HTML, CSS, and Vanilla JavaScript.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Current Phase
Phase 1 - Foundation

## Planned Features
- Ticket CRUD
- Search, filter, sort
- Activity log
- Local persistence
- IndexedDB
- BroadcastChannel sync
- PWA offline support

## Run Locally
Just open `index.html` in the browser.

## Git Workflow
- `main` = stable code
- feature branches for every task
- small commits only
